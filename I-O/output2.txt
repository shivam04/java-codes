THE JAVA.IO PACKAGE CONTAINS NEARLY EVERY CLASS YOU MIGHT EVER NEED TO PERFORM INPUT AND OUTPUT (I/O) IN ,JAVA. ALL THESE STREAMS REPRESENT AN INPUT SOURCE AND AN OUTPUT DESTINATION. THE STREAM IN THE JAVA.IO PACKAGE ,SUPPORTS MANY DATA SUCH AS PRIMITIVES, OBJECT, LOCALIZED CHARACTERS, ETC. ,A STREAM CAN BE DEFINED AS A SEQUENCE OF DATA. THE INPUTSTREAM IS USED TO READ DATA FROM A SOURCE AND THE ,OUTPUTSTREAM IS USED FOR WRITING DATA TO A DESTINATION. ,JAVA PROVIDES STRONG BUT FLEXIBLE SUPPORT FOR I/O RELATED TO FILES AND NETWORKS BUT THIS TUTORIAL COVERS VERY BASIC ,FUNCTIONALITY RELATED TO STREAMS AND I/O.
